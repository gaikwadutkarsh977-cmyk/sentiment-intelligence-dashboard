# Sentiment Intelligence Dashboard

Enterprise-level review sentiment analytics dashboard built using Streamlit and Machine Learning.

## Features

- Upload CSV file
- Automatic sentiment prediction
- Interactive Bar & Donut charts
- KPI Metrics with percentage
- Review filtering
- CSV export
- Clean professional UI

## Tech Stack

- Python
- Streamlit
- Scikit-learn
- Plotly
- Pandas

## How to Run

```bash
pip install -r requirements.txt
streamlit run app.py
